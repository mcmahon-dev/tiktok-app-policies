Privacy Policy

Quiz Generator — TikTok Integration

This Privacy Policy explains how Quiz Generator ("the App", "we", "us") handles data accessed through the TikTok API. The App is a personal automation tool used solely by its owner to manage and analyse content on a single TikTok account.

1. Who This Policy Applies To

The App is not available to the public and has no end users other than its owner. This policy is published to satisfy TikTok's platform requirements for registered developer applications.

2. Data We Access via the TikTok API

The App requests the following TikTok API scopes:

  • user.info.basic — to verify the identity of the authenticated account during the OAuth flow.
  • video.publish — to upload and publish video content to the owner's TikTok account.
  • video.list — to read performance metrics (view count, like count, comment count, share count) for videos posted by the owner's account.

No data belonging to any TikTok user other than the account owner is ever accessed, requested, or stored.

3. How We Use This Data

Data accessed via the TikTok API is used exclusively for:

  • Automating the upload of video content to the owner's own TikTok account.
  • Collecting and storing video performance metrics in a private local database for internal analytics and content optimisation.

The data is never used for advertising, profiling, or any commercial purpose beyond the owner's personal content analysis.

4. Data Storage

All data retrieved from the TikTok API is stored in a private SQLite database on infrastructure controlled solely by the App owner (GitHub Actions and local storage). This database is not publicly accessible and is not shared with any third party.

TikTok API credentials (access tokens, client keys) are stored as encrypted secrets in the owner's private GitHub repository and are never logged or exposed.

5. Data Sharing

We do not sell, rent, trade, or otherwise share any data with third parties. No analytics platforms, advertising networks, or data brokers receive any information derived from TikTok API access.

6. Data Retention

Metric snapshots are retained indefinitely in the private database for longitudinal performance analysis. Access tokens are refreshed periodically and old tokens are discarded. You may request deletion of any stored data by contacting us at the address below.

7. Your Rights

As this App accesses only the owner's own account data, there are no third-party data subjects. If TikTok users have questions about how TikTok itself handles their data, they should refer to TikTok's Privacy Policy.

8. Security

We take reasonable technical measures to protect API credentials and stored data, including encrypted secret storage, private repositories, and restricted access controls on all infrastructure.

9. Changes to This Policy

We may update this policy as the App's functionality changes. The "Last updated" date below will reflect any revisions.
